# gnome-nautilus (Custom Fork)

Personal fork of GNOME Nautilus (Files) v50 with custom modifications.

## Tech Stack

- **Language:** C11
- **Build:** Meson
- **License:** GPL-3.0-or-later
- **Extension API:** libnautilus-extension 4.1

## Build

```bash
meson setup build
meson compile -C build
```

See the upstream [GNOME Nautilus](https://gitlab.gnome.org/GNOME/nautilus) project for full documentation.
